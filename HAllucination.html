MEDIUM

DESIGN ELEMENT
Point, Line, Plane & Shape

DESCRIPTION
These are the artworks I draw with technical drawing pen, these are all designed by myself. Technical drawing pen is one of the tools I like to use for drawing, this is because it can make my artwork stand out and let my works draw more details.

I used some design elements that I learned in semester one to create it such as line, point, dots, plane and shape.

I like to draw something about hallucination, it can show my negative emotions.I also really love to draw something "weird" like monster or ghost. I think they are very cute.
