Aqua Font

TITLE
Aqua font

MEDIUM
Charcoal and Soft Pastel

SIZE
29.7 x 42 cm (A3 Size)

DRAWING TECHNIQUE
Smudge, Blending + Erasing, Scumble

DESCRIPTION
This idea is inspired by seafood feast that cooked by my mom.I decided to design a font with sea theme and named it Aqua Font.This is because there are many things that I can draw in this theme.

The elements I use in this design are colour,line and shape.I have used warm and cool colour to design this font to makes my design looks more interesting.I also design something that only exists in legends such as mermaid in my font.

I like the sea so much,because the sea is mysterious.There are many unknown things in the sea,which makes people very curious.I think this font design can used for logo or poster of aquarium because my theme is about sea.
